#  Student Performance Predictor & Personalized Academic Advisor  
### (System is based on data of VIT  Bhopal students.)

---

##  Overview

This project is a Machine Learning-based system designed to help students of **VIT Bhopal** analyze and improve their academic performance.

It predicts student marks based on key academic factors and provides **GPA estimation + personalized suggestions** to help students improve their performance.

---

##  Problem Statement

Many students struggle to understand:
- How their daily habits affect academic performance  
- Why their GPA is low  
- What specific actions they should take to improve  

This leads to:
- Poor academic planning  
- Low GPA  
- Increased stress  

---

##  Solution

This project solves the problem by:

- Predicting student marks using Machine Learning  
- Converting marks into GPA (based on VIT grading system)  
- Providing **personalized academic suggestions**  
- Giving students actionable insights  

---

##  Features

-  Predicts marks using ML model  
-  Converts marks to GPA (VIT system)  
-  Gives personalized improvement suggestions  
-  Interactive UI using Streamlit  
-  Real-time prediction  

---

##  How It Works

1. User enters:
   - Study hours  
   - Attendance  
   - Assignment score  
   - Previous GPA  

2. ML model predicts marks  

3. Marks are converted into:
   - Grade  
   - GPA  

4. System generates **personalized suggestions**
