# Student Performance Predictor & Advisor

##  Overview
This project predicts student marks based on study habits and provides GPA along with personalized suggestions.

##  Features
- Predicts marks using ML
- Converts marks to GPA (VIT system)
- Gives personalized improvement suggestions
- Simple UI using Streamlit

##  Tech Stack
- Python
- Scikit-learn
- Streamlit

##  How to Run

```bash
pip install -r requirements.txt
python train_model.py
streamlit run app.py
